Class Detail

HornPatternConverter()
Used to create new HornPatternConverter instances, thus:
var converter = new HornPatternConverter( {horn: aHornInstance}); .


The HornPatternConverter is a utility class that binds itself to a Horn instance and helps with model type conversions.

Reusable converter functions are registered to execute when Horn conversion operations involve property paths that match given criteria.

Parameters:
{Horn} args.horn
the Horn instance to bind to

  • pattern(pattern, converterName)
    Add a pattern, bound to a given named converter.

    When Horn initiates a convert operation for a path that matches 'pattern', the converter (previously registered) under 'converterName' will be called upon to perform the conversion operation.

    Patterns registered with this function are normal Horn property paths that may or may not include the meta-character '*' that will match any sequence of characters.

    Parameters:
    {String} pattern
    a Horn property path with optional wildcard '*' characters
    {String} converterName
    the name of the converter that will handle conversions for the given property path
  • regexPattern(pattern, converterName)
    Add a regex pattern, bound to a given named converter.

    Parameters:
