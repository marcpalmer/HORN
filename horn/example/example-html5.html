This example demonstrates using the HORN specification plus the jQuery reference implementation to extract data from vanilla HTML, in a way similar to microformats. The information on the left, the book list, is straight HTML from the server.

On the right is the data that the JS HORN implementation extracted from the page, in a JSON-like presentation purely to show the model that is extracted.

You can change values in the panel on the right and click the update button to reverse the process and push the JS data model back into the original DOM nodes. This includes type coercions and data conversions in both directions.
